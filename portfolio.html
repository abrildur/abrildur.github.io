MexIA

It is a project for the Hackathon organized by Innovaccion Virtual Microsoft.

The main idea of making this platform was to share the origin of one of our most beloved and famous tradition around the world "Dia de Muertos" to foreign people or even to fellow mexicans whom are willing to know more about our roots using Artificial Intelligence with Microsoft Technologies.

Link to the Github repository where you can find more information about the project among it's source code.

Languages and Frameworks: JavaScript, CSS, Python, and Uikit
Technologies: Azure QnA Maker, Azure Bot, Azure Custom Vision
